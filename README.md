# ModernAirlineRetailsDemo
A weekend-scale demo project showcasing Modern Airline Retailing concepts (Offers &amp; Orders) with Java Spring Boot microservices, event-driven architecture, Kafka, Saga pattern, Outbox, and CQRS.

✅ Offer üretimi
✅ Dinamik fiyat hesaplama
✅ Order oluşturma
✅ Ödeme süreci
✅ Saga orkestrasyonu
✅ Outbox + Kafka event yayılımı
✅ CQRS ile read modeli
